Module: config/database

Configuración y conexión a la base de datos MongoDB utilizando Mongoose. Este módulo se encarga de establecer la conexión con MongoDB y manejar posibles errores.

(async, inner) connectToDatabase() → {Promise.<void>}

Conecta a la base de datos MongoDB utilizando la URI proporcionada en las variables de entorno. Maneja la conexión con Mongoose y proporciona logs detallados para el estado de la conexión.
