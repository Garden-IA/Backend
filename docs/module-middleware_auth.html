(static) verifyToken(req, res, next) → {void}

Middleware para verificar el token JWT en las solicitudes.

Este middleware extrae el token del encabezado 'Authorization', verifica su validez y añade la información del usuario decodificada al objeto de solicitud (req.user).

El objeto de solicitud.
