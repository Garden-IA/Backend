(static) validateRegisterInput(data) → {Object}

Valida la entrada para el registro de un usuario.

Esta función verifica que se hayan proporcionado un nombre de usuario y una contraseña. Si faltan alguno de estos campos, se devuelve un objeto de errores.

Los datos de entrada para la validación.
